# Solar System Simulator

A simple solar system simulation built with Unity.
The simulation visualizes planets orbiting around the Sun in real time, along with a small top-down view displayed in the lower-left corner of the screen for an overall system overview.

## Features
Sun-centered solar system model
Top-down mini view of the entire system
3D visualization
